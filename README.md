# SnapCurrency 💱

A responsive currency converter built with **HTML**, **CSS**, and **JavaScript**. SnapCurrency provides live currency conversion using real-time exchange rates and automatically falls back to built-in offline rates for common currencies when a network connection is unavailable.

---

## Overview

SnapCurrency is a lightweight web application designed to make currency conversion fast, simple, and reliable. It integrates with a live exchange rate API to provide up-to-date conversion rates while ensuring the application remains functional offline through predefined fallback rates.

The project also includes several user-friendly features such as currency search, favorites, conversion history, theme switching, and input validation, making it a practical front-end project demonstrating modern JavaScript concepts.

---

## Features

- Live currency conversion using exchange rate API
- Offline fallback rates for common currencies
- Searchable currency selection
- Swap currencies with a single click
- Save favorite currencies
- Conversion history
- Light/Dark theme toggle
- Input validation
- Responsive design for desktop and mobile devices
- Clean and intuitive user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API
- FlagCDN

## Installation

### Clone the repository

```bash
git clone https://github.com/UserAdityaKhawale/SnapCurrency.git
```

### Navigate to the project directory

```bash
cd SnapCurrency
```

### Open the application

Simply open the `index.html` file in your preferred web browser.

No additional installation or build tools are required.

---

## Usage

1. Select the source currency.
2. Select the target currency.
3. Enter the amount to convert.
4. View the converted value instantly.
5. Use the search feature to quickly find currencies.
6. Save frequently used currencies as favorites.
7. Toggle between light and dark themes according to your preference.
8. Access previous conversions through the conversion history.
